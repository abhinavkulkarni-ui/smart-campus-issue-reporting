Frequently Asked Questions

How does the AI decide priority?

We use a keyword-based NLP model. Words like "Fire", "Spark", or "Gas" trigger a High Priority alert. Maintenance words like "Dripping" trigger Medium/Low priority.

Where is my data stored?

For this prototype, data is stored locally in your browser (LocalStorage). In the production version, we use AWS DynamoDB for secure cloud storage.

Is this system secure?

Yes. The Admin Dashboard is password-gated. Future updates will include JWT Authentication via AWS Cognito.
